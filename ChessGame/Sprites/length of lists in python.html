    Method calling another method in the same class

    Can a method in class call another method in that same class?

    example:
    python Code:
     
def open_file(self, obj):
		""" Opens a file """
		file_open = gtk.FileChooserDialog(title="Open", action=gtk.FILE_CHOOSER_ACTION_OPEN,
					buttons=(gtk.STOCK_CANCEL, gtk.RESPONSE_CANCEL, gtk.STOCK_OPEN, gtk.RESPONSE_OK))
 
		chosen_file = ""
		# if 'open' was clicked, get the name of the file. if cancel was clicked, chosen_file = ''
		if file_open.run() == gtk.RESPONSE_OK:
			chosen_file = file_open.get_filename()
			# call show(chosen_file) <--- trying to do this
            # which would call the method below
